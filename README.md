# Portfolio Website Project

Questo progetto è un sito portfolio creato con l'obiettivo principale di praticare HTML, CSS, JavaScript, Git e GitHub. Il sito è stato sviluppato come parte del mio percorso di apprendimento nel campo dello sviluppo web e della gestione dei progetti con Git.

## Obiettivi

Il principale obiettivo di questo progetto è quello di migliorare le mie competenze pratiche in:

- **HTML:** Creare una struttura semantica e accessibile per il sito portfolio, includendo sezioni per la mia biografia, progetti, skills e modi per contattarmi.
- **CSS:** Applicare stili accattivanti e responsivi per rendere il sito esteticamente gradevole su diverse dimensioni di schermo e dispositivi.
- **JavaScript:** Implementare funzionalità interattive e dinamiche, come la navigazione a schede, la visualizzazione delle immagini dei progetti in modalità lightbox e l'invio di messaggi tramite un modulo di contatto.
- **Git e GitHub:** Utilizzare Git per gestire il controllo delle versioni del codice sorgente e collaborare con altri sviluppatori tramite GitHub.
